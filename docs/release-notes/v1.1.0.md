# Office PDF Binder v1.1.0

Release date: June 14, 2026

### 改善

- ウィンドウ位置とサイズを保存し、画面外に出る場合は自動補正するようにしました。
- Word / Excel / PowerPoint のサムネイル表示を改善しました。
- PDFページの一覧表示で、ページ番号（.P1 など）が省略されにくいようにしました。
- 長押しドラッグによるページ並び替えの操作性を改善しました。
- ドラッグ&ドロップ移動時の挿入位置表示を見やすくしました。
- READMEに画面例を追加しました。
- アンインストール時にユーザー設定フォルダを削除するようにしました。

### 修正

- Officeファイル変換失敗時に複数のエラーダイアログが表示される問題を修正しました。
- インストール版でアプリのウィンドウアイコンが反映されない場合がある問題を修正しました。

### インストール時の注意

- v1.0.0 をインストール済みの場合は、先に v1.0.0 をアンインストールしてから v1.1.0 をインストールしてください。
- 本アプリは未署名アプリのため、環境によっては Windows SmartScreen などの警告が表示される場合があります。
- Word / Excel / PowerPoint ファイルをPDF変換するには、Microsoft Office が必要です。
