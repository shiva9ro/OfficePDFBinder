# Office PDF Binder v1.2.0

Release date: June 21, 2026

## English

- Added English user interface support
- Added English and Japanese manuals
- Added English and Japanese installer support
- Added a portable edition that requires no installation and minimizes settings and log writes to the PC
- Improved automated tests and build workflows
- Improved source-code packaging for AGPL compliance
- Updated GitHub issue forms for English and Japanese users

Microsoft Office is required to convert Word, Excel, and PowerPoint files.
