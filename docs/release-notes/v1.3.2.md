# Office PDF Binder v1.3.2

Release date: July 4, 2026

[日本語](#user-content-japanese)

## English

### Highlights

- Added automatic retries for Office conversion failures during both GUI and
  command-line subfolder batch processing. Word, Excel, and PowerPoint
  conversions are attempted up to three times with a one-second delay.
- Kept regular GUI saves unchanged; automatic retries are limited to subfolder
  batch processing.
- Improved Office instance handling so that Word and Excel use dedicated
  automation instances and an existing shared PowerPoint session is not closed.
- Added retry recovery counts to the existing batch CSV log.

### Downloads

- `OfficePDFBinder_Setup_1.3.2.exe`: Windows installer
- `OfficePDFBinder_Portable_1.3.2.zip`: Portable edition

### Notes

- Microsoft Office is required to convert Word, Excel, and PowerPoint files.
- Password-protected or damaged Office files may still require user interaction
  or fail after all retry attempts.
- The distributed executables are currently unsigned. Windows SmartScreen or
  antivirus software may display a warning.

## 日本語

### 主な変更点

- GUIおよびコマンドラインのサブフォルダ一括処理でOffice変換に失敗した場合、
  1秒間隔で最大2回自動的に再試行するようにしました。初回を含めて最大3回
  試行します。
- 通常のGUI保存は従来どおりとし、自動再試行はサブフォルダ一括処理だけに
  限定しました。
- WordとExcelで専用の自動操作用インスタンスを使用し、ユーザーが開いている
  PowerPointを終了しないよう、Officeインスタンスの扱いを改善しました。
- 再試行によって変換に成功した件数を、既存の一括処理CSVログへ記録するように
  しました。

### ダウンロード

- `OfficePDFBinder_Setup_1.3.2.exe`：Windowsインストーラー
- `OfficePDFBinder_Portable_1.3.2.zip`：ポータブル版

### 注意事項

- Word、Excel、PowerPointの変換にはMicrosoft Officeが必要です。
- パスワード保護または破損したOfficeファイルでは、ユーザー操作が必要になるか、
  すべての再試行後も変換に失敗する場合があります。
- 配布する実行ファイルには現在コード署名がありません。Windows SmartScreenや
  ウイルス対策ソフトが警告を表示する場合があります。
