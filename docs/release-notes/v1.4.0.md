# Office PDF Binder v1.4.0

Release date: July 12, 2026

[日本語](#user-content-japanese)

## English

### Highlights

- Added LibreOffice-based Word, Excel, and PowerPoint to PDF conversion as an
  alternative to Microsoft Office automation.
- Added Office conversion engine selection: Auto, Prefer Microsoft Office, and
  Prefer LibreOffice.
- Added support for WebP, TIFF, HEIC/HEIF, and SVG files when creating a merged
  PDF.
- Added multi-page TIFF handling so each image frame is added as a separate
  page.
- Made save/error result dialogs copyable so conversion errors can be copied for
  troubleshooting.
- Improved toolbar layout so the primary save action remains visible at smaller
  window widths.
- Reorganized settings menu items to keep related image, annotation, and Office
  conversion options together.

### Downloads

- `OfficePDFBinder_Setup_1.4.0.exe`: Windows installer
- `OfficePDFBinder_Portable_1.4.0.zip`: Portable edition

### Notes

- Office document conversion requires either Microsoft Office or LibreOffice.
- Microsoft Office generally provides the best fidelity for Office documents.
  LibreOffice is useful on environments where Microsoft Office is unavailable.
- HEIC/HEIF support is provided through `pi-heif`.
- The distributed executables are currently unsigned. Windows SmartScreen or
  antivirus software may display a warning.

## 日本語

### 主な変更点

- Microsoft Officeの自動操作に加えて、LibreOfficeによるWord、Excel、
  PowerPointのPDF変換に対応しました。
- Office変換エンジンの設定を追加しました。選択肢は「自動」、
  「Microsoft Officeを優先」、「LibreOfficeを優先」です。
- WebP、TIFF、HEIC/HEIF、SVGファイルをPDF結合に追加できるようにしました。
- 複数ページTIFFは、各フレームを個別ページとして取り込むようにしました。
- 保存完了やエラーのダイアログをコピー可能にし、変換失敗理由をそのまま
  コピーできるようにしました。
- 小さいウィンドウ幅でも主操作の保存ボタンが見えやすいよう、
  ツールバーの並びと表示文言を整理しました。
- 設定メニューを整理し、画像、注釈、Office変換に関する項目を近くに
  まとめました。

### ダウンロード

- `OfficePDFBinder_Setup_1.4.0.exe`：Windowsインストーラー
- `OfficePDFBinder_Portable_1.4.0.zip`：ポータブル版

### 注意事項

- Office文書の変換には、Microsoft OfficeまたはLibreOfficeのどちらかが
  必要です。
- Office文書の再現性は、一般的にはMicrosoft Officeの方が高くなります。
  LibreOfficeは、Microsoft Officeを利用できない環境で有効です。
- HEIC/HEIF対応には `pi-heif` を使用しています。
- 配布する実行ファイルには現在コード署名がありません。Windows SmartScreenや
  ウイルス対策ソフトが警告を表示する場合があります。
