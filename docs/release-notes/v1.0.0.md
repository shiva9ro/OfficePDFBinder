# Office PDF Binder v1.0.0

Release date: May 18, 2026

## 初回リリース

Office PDF Binder v1.0.0 の初回公開版です。

### 主な機能

- PDF / Word / Excel / PowerPoint ファイルの読み込み
- ページ単位の並べ替え、削除、回転
- ドラッグ&ドロップによるファイル追加とページ移動
- 自動しおり、既存PDFしおりの読み込み、手動しおり編集
- ヘッダー・フッター、日付、ページ番号の追加
- 選択ページのPDF / JPEG書き出し
- Windows右クリックメニュー連携

### 動作環境

- Windows 10 / 11 64bit
- Officeファイル変換には Microsoft Office が必要です

### インストール

下の `OfficePDFBinder_Setup_1.0.0.exe` をダウンロードして実行してください。
