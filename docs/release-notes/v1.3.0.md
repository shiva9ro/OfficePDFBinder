# Office PDF Binder v1.3.0

Release date: June 27, 2026

[日本語](#user-content-japanese)

## English

### Highlights

- Added command-line batch processing for automation from PowerShell and BAT files.
- Added batch creation of one merged PDF for each direct subfolder.
- Added PNG, JPEG, and BMP images as A4 PDF pages.
- Added insertion of blank A4 portrait pages.
- Added selectable JPEG export resolution: 96, 150, 300, or 600 dpi.
- Added an option to remove comments, highlights, drawings, and other PDF annotations from output PDFs.
- Added an option to suppress Word comments and tracked changes, and Excel comments during PDF conversion.
- Added an option to prevent small images from being enlarged beyond their 96 dpi reference size.
- Added UTF-8 BOM CSV logs for subfolder batch processing.
- Reorganized build and packaging support files.
- Expanded the automated test suite to 100 tests.

### Command-line batch mode

The CLI accepts separate input and output folders and uses the same batch
processing implementation as the GUI.

```text
--batch-subfolders
--input-root <path>
--output-root <path>
```

Options are available for bookmarks, PDF annotation removal, Word/Excel review
information suppression, image upscaling prevention, and overwriting existing
PDFs. The CLI returns a nonzero exit code when any item is Warning, Skipped, or
Error.

### Downloads

- `OfficePDFBinder_Setup_1.3.0.exe`: Windows installer
- `OfficePDFBinder_Portable_1.3.0.zip`: Portable edition

### Notes

- Microsoft Office is required to convert Word, Excel, and PowerPoint files.
- Source Office documents and source PDFs are not modified by the review
  information and PDF annotation suppression options.
- The distributed executables are currently unsigned. Windows SmartScreen or
  antivirus software may display a warning.

### CLI一括処理

CLIでは入力親フォルダと出力フォルダを別々に指定し、GUIと同じ一括処理を使用します。

```text
--batch-subfolders
--input-root <path>
--output-root <path>
```

しおり、PDF注釈除去、Word/Excelのレビュー情報抑制、画像拡大抑制、既存PDFの上書きをオプションで指定できます。Warning、Skipped、Errorが1件でもある場合、CLIは0以外の終了コードを返します。

- レビュー情報抑制とPDF注釈除去では、元のOffice文書や元PDFを変更しません。
- 配布する実行ファイルには現在コード署名がありません。Windows SmartScreenやウイルス対策ソフトが警告を表示する場合があります。
