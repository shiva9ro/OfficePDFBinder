# Office PDF Binder v1.3.1

Release date: June 29, 2026

[日本語](#user-content-japanese)

## English

### Highlights

- Updated the application runtime from Python 3.12 to Python 3.13.
- Updated PyMuPDF, PySide6, Pillow, qtawesome, pywin32, Nuitka, Markdown,
  and pytest.
- Changed the Windows build compiler to Clang-cl for reliable compilation of
  the large PyMuPDF wrapper.
- Improved application startup performance with the updated runtime and
  dependencies.
- Added repeatable Python environment benchmarks and JSON build-time records.

This is a maintenance release. The document editing and batch-processing
features are unchanged from v1.3.0.

### Downloads

- `OfficePDFBinder_Setup_1.3.1.exe`: Windows installer
- `OfficePDFBinder_Portable_1.3.1.zip`: Portable edition

### Notes

- Microsoft Office is required to convert Word, Excel, and PowerPoint files.
- The distributed executables are currently unsigned. Windows SmartScreen or
  antivirus software may display a warning.

## 日本語

### 主な変更点

- アプリの実行環境をPython 3.12からPython 3.13へ更新しました。
- PyMuPDF、PySide6、Pillow、qtawesome、pywin32、Nuitka、Markdown、
  pytestを更新しました。
- 大規模なPyMuPDFラッパーを安定してコンパイルするため、Windows版の
  ビルドコンパイラをClang-clへ変更しました。
- 実行環境と依存ライブラリの更新により、アプリの起動速度を改善しました。
- Python環境の性能比較ツールと、JSON形式のビルド時間記録を追加しました。

本リリースは保守更新です。文書編集機能と一括処理機能はv1.3.0から変更していません。

### ダウンロード

- `OfficePDFBinder_Setup_1.3.1.exe`：Windowsインストーラー
- `OfficePDFBinder_Portable_1.3.1.zip`：ポータブル版

### 注意事項

- Word、Excel、PowerPointの変換にはMicrosoft Officeが必要です。
- 配布する実行ファイルには現在コード署名がありません。Windows SmartScreenや
  ウイルス対策ソフトが警告を表示する場合があります。
